# uv

## 常用命令

* `uv --version`
* `uv --help`
* `uv self update`

### 项目初始化

* `uv init`
  * 使用当前目录初始化环境
* `uv init --python 3.13 my-project`
  * 指定 python版本 和 项目名 初始化环境

### 依赖管理

* `uv add <包名>`
  * 安装生产依赖（写入 pyproject.toml + 锁定版本）
* `uv add --dev <包名>`
  * 安装开发依赖。
* `uv update <包名>`
  * 更新指定依赖到最新兼容版本
* `uv remove <包名>`
  * 移除依赖
* `uv sync`
  * 同步依赖

### CI/CD

* `uv build`
  * 编译
